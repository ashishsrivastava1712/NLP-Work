🧠 NLP CONCEPTS IMPLEMENTED
───────────────────────────────────────────────────────────────────────────────────────────────────────

✅ TOKENIZATION (splitting text into words)
   Location: src/text_preprocessor.py
   Implementation: NLTK word_tokenize()
   Example: "What is Python?" → ["What", "is", "Python", "?"]

✅ STOPWORD REMOVAL (removing common non-informative words)
   Location: src/text_preprocessor.py
   Implementation: NLTK stopwords
   Example: ["What", "is", "Python"] → ["Python"]
   (Removes: "what", "is", "a", "the", etc.)

✅ TEXT PREPROCESSING (complete cleaning pipeline)
   Location: src/text_preprocessor.py
   Implementation: 4-step pipeline
   Steps: Lowercase → Remove special chars → Tokenize → Remove stopwords

✅ TF-IDF VECTORIZATION (converting text to numerical vectors)
   Location: src/chatbot.py
   Implementation: sklearn TfidfVectorizer
   Purpose: Represent importance of each word as numbers
   Example: "machine learning" → [0.45, 0.89, 0.12, ...]

✅ COSINE SIMILARITY (measuring text similarity)
   Location: src/chatbot.py
   Implementation: sklearn cosine_similarity
   Purpose: Find best matching Q&A pair
   Range: 0 (different) to 1 (identical)
   Example: User query vs 20 Q&A pairs, pick highest
