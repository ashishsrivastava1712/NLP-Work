🧠 NLP CONCEPTS EXPLAINED
════════════════════════════════════════════════════════════════════════════════

Concept 1: TOKENIZATION
  What: Breaking text into words
  Where: src/text_preprocessor.py (line 52)
  Example:
    Input: "What is Python?"
    Output: ["What", "is", "Python", "?"]
  Tool: NLTK word_tokenize()
  Why: Computers work with individual units, not full sentences

Concept 2: STOPWORD REMOVAL
  What: Filtering common non-informative words
  Where: src/text_preprocessor.py (line 60)
  Example:
    Before: ["What", "is", "Python"]
    After: ["Python"]  (removed "What", "is")
  Removed words: "the", "is", "and", "a", "an", "to", etc.
  Why: These words don't carry meaning, just add noise

Concept 3: TEXT PREPROCESSING
  What: Complete cleaning pipeline
  Where: src/text_preprocessor.py (class TextPreprocessor)
  Steps:
    1. Lowercase: "What IS PYTHON?" → "what is python?"
    2. Clean special chars: "Hello!!!" → "Hello"
    3. Tokenize: "What is Python?" → ["What", "is", "Python", "?"]
    4. Remove stopwords: ["What", "is", "Python"] → ["Python"]
  Why: Standardize text for consistent processing

Concept 4: TF-IDF VECTORIZATION
  What: Converting text to numerical vectors
  Where: src/chatbot.py (line 50)
  Tool: sklearn TfidfVectorizer
  Formula: TF-IDF = (Word Frequency) × (Inverse Document Frequency)
  Example:
    Text: "machine learning"
    Vector: [0.45, 0.89, ...]  (numerical representation)
  Why: Computers need numbers to do math, not text
  Importance: Words that appear often AND unique get higher scores

Concept 5: COSINE SIMILARITY
  What: Measuring how similar two texts are
  Where: src/chatbot.py (line 55)
  Tool: sklearn cosine_similarity
  Range: 0 (completely different) to 1 (identical)
  Example:
    User query vs Question 1: 0.95 (95% similar) ← Best match!
    User query vs Question 2: 0.30 (30% similar)
    User query vs Question 3: 0.10 (10% similar)
  Why: Find which Q&A pair is most similar to user's question

ALL TOGETHER:
  1. User types: "Tell me about machine learning"
  2. Tokenize: ["Tell", "me", "about", "machine", "learning"]
  3. Remove stopwords: ["machine", "learning"]
  4. Convert to TF-IDF vector
  5. Compare with 20 Q&A pair vectors using cosine similarity
  6. Find highest similarity: matches "What is machine learning?" (0.95)
  7. Return that answer + 95% confidence score
