                    🧠 NLP CONCEPTS EXPLAINED BY FILE
════════════════════════════════════════════════════════════════════════════════

TOKENIZATION
  What: Breaking text into individual words
  Where: src/text_preprocessor.py (line ~55)
  Example: "What is Python?" → ["What", "is", "Python", "?"]
  Tool: NLTK word_tokenize()

STOPWORD REMOVAL
  What: Removing common non-informative words
  Where: src/text_preprocessor.py (line ~60)
  Example: ["What", "is", "Python"] → ["Python"]
  Tool: NLTK stopwords
  Common words removed: "the", "is", "and", "a", "to", etc.

TEXT PREPROCESSING (Complete Pipeline)
  What: Cleaning text for NLP processing
  Where: src/text_preprocessor.py (preprocess method)
  Steps:
    1. Lowercase
    2. Remove special characters
    3. Tokenize
    4. Remove stopwords
  Example:
    Input: "What IS Python???"
    Output: "python"

TF-IDF VECTORIZATION
  What: Converting text into numerical vectors
  Where: src/chatbot.py (line ~50)
  Purpose: Represent importance of each word
  Tool: sklearn TfidfVectorizer
  Example:
    Text 1: "python programming"
    Vector 1: [0.5, 0.3, 0.8, ...]
    Text 2: "machine learning"
    Vector 2: [0.1, 0.9, 0.2, ...]

COSINE SIMILARITY
  What: Measuring how similar two vectors are
  Where: src/chatbot.py (line ~55)
  Purpose: Find best matching Q&A pair
  Tool: sklearn cosine_similarity
  Range: 0 (different) to 1 (identical)
  Example:
    User query vector similarity with:
      Q1: 0.95 (95% match) ← Best match!
      Q2: 0.30 (30% match)
      Q3: 0.15 (15% match - too low)

    ↓
  Find Best Match (src/chatbot.py, line ~60)

  Return Answer + Confidence
