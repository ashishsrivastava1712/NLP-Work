src/text_preprocessor.py
────────────────────────────────────────────────────────────────
Purpose: Text cleaning and preprocessing

Contains: TextPreprocessor class

Methods:
  • __init__() ..................... Initialize with stopwords
  • clean_text(text) ............... Remove special characters
  • preprocess(text) ............... Complete preprocessing pipeline

What it does (step-by-step):
  1. Lowercase text
     "Python IS GREAT" → "python is great"
  
  2. Remove special characters
     "Hello!!! $$$" → "Hello"
  
  3. Tokenization (split into words)
     "What is Python?" → ["What", "is", "Python", "?"]
  
  4. Remove stopwords (common words)
     ["What", "is", "Python"] → ["Python"]
  
  5. Join back
     ["Python"] → "python"

Lines of code: ~80
Complexity: Low (text manipulation)
Used by: chatbot.py


src/qa_dataset.py
────────────────────────────────────────────────────────────────
Purpose: Q&A knowledge base for the chatbot

Contains: QA_DATASET dictionary (20 Q&A pairs)

Topics covered:
  ✓ Python & Programming (4 pairs)
  ✓ Machine Learning (4 pairs)
  ✓ Natural Language Processing (5 pairs)
  ✓ Information Retrieval & Similarity (2 pairs)
  ✓ AI & Chatbots (3 pairs)
  ✓ Data Science (2 pairs)

Functions:
  • get_dataset() .................. Returns full dataset
  • dataset_size() ................. Returns number of Q&A pairs
  • get_all_questions() ............ Returns all questions
  • get_all_answers() .............. Returns all answers

How to customize:
  Edit QA_DATASET dictionary and add new pairs:
  
  QA_DATASET = {
      "Existing question": "Existing answer",
      "Your new question": "Your new answer",  ← Add here
  }

Lines of code: ~110
Complexity: Very Low (just data)
Format: Python dictionary


src/chatbot.py
────────────────────────────────────────────────────────────────
Purpose: Core chatbot logic using TF-IDF and cosine similarity

Contains: RetrievalChatbot class

Methods:
  • __init__(qa_dataset) .......... Initialize with Q&A pairs
  • find_best_match(query, threshold) ... Find similar Q&A
  • chat(query) ................... Process and respond
  • print_response(result) ........ Format console output

How it works:
  1. User asks: "What is machine learning?"
  2. Preprocess: "machine learning"
  3. TF-IDF vectorization: Convert to numbers
  4. Compare with all 20 Q&A pairs
  5. Find highest similarity: 0.95 (95% match)
  6. Return answer + confidence

Key Features:
  ✓ TF-IDF vectorization (sklearn)
  ✓ Cosine similarity matching (sklearn)
  ✓ Configurable threshold (default 0.3 = 30%)
  ✓ Fallback for unknown queries
  ✓ Similarity scores displayed

Lines of code: ~120
Complexity: Medium
Used by: main.py, streamlit_chatbot.py

🆘 QUICK HELP
════════════════════════════════════════════════════════════════════════════════

"Module not found error"
→ Run: pip install -r requirements.txt

"Permission denied"
→ Right-click terminal, Run as Administrator

"Port already in use" (for Streamlit)
→ Close other Streamlit sessions or use: 
  streamlit run ui/streamlit_chatbot.py --server.port 8502

"Low similarity scores"
→ Edit src/chatbot.py, change threshold=0.3 to 0.2

"NLTK data missing"
→ Run: python -c "import nltk; nltk.download('punkt'); nltk.download('stopwords')"

"Want to add more questions"
→ Edit src/qa_dataset.py and add to QA_DATASET dictionary

"Want to change web UI"
→ Edit ui/streamlit_chatbot.py
